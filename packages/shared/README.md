# Shared

Este diretório contém código, componentes e recursos compartilhados entre diferentes partes do projeto.

**Propósito**: Centralizar utilitários, constantes, tipos compartilhados, componentes reutilizáveis e outras funcionalidades que são usadas por múltiplas aplicações.

Use este diretório para evitar duplicação de código e manter a consistência entre as diferentes aplicações do projeto.
